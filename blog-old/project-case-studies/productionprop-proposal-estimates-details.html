ProductionProp.com: Transforming Film Production with E-commerce

Introduction

ProductionProp.com is an innovative e-commerce platform dedicated to the film production industry. Designed specifically for prop houses, film set designers, and production companies, this platform aims to simplify the process of sourcing and renting essential props and materials for film projects.

Product Overview

ProductionProp.com serves as a marketplace allowing users to purchase or rent props, furniture, clothing, and other essential materials for movie sets. Its standout feature is the interactive scene configurator, enabling designers to visualize and configure scenes before finalizing their orders.

Key Features

  • E-commerce for Purchase & Rent: Seamless cart management, checkout process, and reservation requests.
  • Interactive Scene Configurator: Plan and visualize scenes using the products listed.
  • City and Date Selector: Filter availability by city and event date, launching exclusively in Los Angeles.
  • Theme-Based Filtering: Search products based on movie themes like Horror, History, and Space.
  • Integrated Payment Gateway: Support for multiple payment methods for secure transactions.
  • User Accounts: Designer and vendor accounts with tailored functionality.
  • Admin Dashboard: Tools for managing inventory, featuring products, and monitoring rentals.

Technology Stack

  • Front-end: Next.js for dynamic and responsive user interfaces.
  • Back-end: Nest.js for scalable API development.
  • Database: PostgreSQL for robust data storage.
  • Authentication: Google and email-based login support.
  • Hosting: AWS cloud platform for scalable deployments.

MVP Scope

The MVP version focuses on delivering core functionalities:

  • Infrastructure setup, including deployments and domain mapping.
  • User authentication and account management.
  • Cart management and payment integration.
  • Core e-commerce flows for purchases and rentals.
  • Search, reservations, and admin panel capabilities.

Goals and Objectives

ProductionProp.com seeks to streamline the complexities of set design and prop management in the film industry. The platform aspires to become the go-to marketplace for set designers and prop houses, enhancing efficiency and creativity in film production.
